WINDOWS (Vista/7/8/10/11/Server)

Para acesso s informaes do sistema incluindo os componentes fsicos, aplicaes e demais software utilizaremos a ferramenta Intel System Support Utility (ISSU) for Windows, que permite a gerao de um relatrio completo das configuraes da mquina.

Baixando e executando o ISSU

  1. Clique AQUI e baixe o arquivo intel-ssu-windows.exe;

  2. Execute o arquivo baixado (intel-ssu-windows.exe). Caso surja um dilogo do sistema com a mensagem Deseja permitir que este aplicativo faa alteraes no seu dispositivo?, clique em Sim. Ser aberta a janela do ISSU. Marque a caixa de seleo Everything e depois no boto Scan.

Feito isto, o ISSU realizar a deteco de todos os componentes do computador. Esta etapa pode demorar at 1min.

Visualizando e exportando os resultados para um documento de texto (.txt)

  1. Aps a execuo da etapa anterios, a janela do ISSU ser atualizada e exibir um resumo da informaes do sistema.
  1. No menu dropdown na parte superior esquerda da janela do ISSU estar selecionada a opo Summary View; Clicando neste menu haver outra opo Detailed View. Selecione-a para visualizar as configuraes completas do computador.

  1. Para gerar um relatrio completo das informaes obtidas pelo ISSU, clique no boto Next>, localizado no canto inferir direito da tela.

  2. A seguir, clique no boto Save.

  1. Ser aberta um dilogo padro do Windows para salvar arquivos. D um nome ao arquivo que ser salvo e escolha uma pasta de sua preferncia. Por ltimo clique em Salvar. Ser gerado e salvo um arquivo de texto (.txt) contendo um relatrio detalhado das informaes da mquina.

LINUX (distros baseadas em Debian)

Para acesso s informaes do sistema incluindo os componentes fsicos, aplicaes e demais software em distribuies Linux baseadas em Debian, incluso o prprio Debian, Ubuntu, Linux Mint, Pop!OS entre diversos outros, sugerimos a utilizao do pacote lshw (list hardware). Este pacote de fcil utilizao e se encontra nos repositrios oficiais do Debian e do Ubuntu. Possui verses tambm nos repositrios de outros projetos como o Mandriva, RedHat e OpenSUSE. Descreverems a seguir as etapas para instalao e e execuo do lshw via terminal nas distribuies baseadas em Debian.

Instalao e execuo do pacote o lshw:

Abra o terminal e execute:

  1. Para atualizar o ndice de pacotes:
sudo apt-get update
  1. Para instalar o lshw:
sudo apt-get install lshw
  1. Para executar o lshw e visualizar as configuraes:
sudo lshw

Exportando o relatrio no formato .html como as configuraes da mquina

sudo lshw -html > /home/seu-usuario/nome-do-arquivo.html

A exportao no formato .html facilita a visualizao e extrao dos dados de interesse. No entanto tambm possvel exportao nos formatados de texto simples (.txt), XML (.xml) e JSON (.json), bastando substituir os parmetros -html e a extenso .html

sudo apt-get install lshw-gtk
sudo lshw -X

A instalao do pacote lshw pode ser realizada pela central/loja de aplicativos de todas as distribuies atuais Linux basedas em Debian.
